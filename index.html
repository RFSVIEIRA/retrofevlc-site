Welcome to RetroFeVLC

RetroFeVLC is a powerful fork of RetroFE that replaces GStreamer with VLC, bringing dozens of new features, a smart addon system, and a much more user-friendly experience.

Quick Start Guide

  1. Download the latest Full All Addons Include version
  2. Extract the archive to any folder (e.g. E:\RetroFeVLC)
  3. Run Start.exe
  4. Right-click the Start icon in the System Tray → RetroFeVLC Options → Settings Editor
  5. Right-click again → Controls Editor
  6. Start adding your own game collections

Key Features

Intelligent launcher with System Tray, automatic sync of Favorites & Last Played, JoyToKey, SuperF4 and more.
Lightning-fast game search (F11 by default) with full gamepad support.
Grid menus, per-game layouts, random slideshows, 3D effects, and built-in Options Menu.

About RetroFeVLC

RetroFeVLC is a community-driven fork developed by an enthusiast. Fully open-source under GPL3. The goal is to make RetroFE even better while keeping it free and open for everyone.
