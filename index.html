What is RetroFeVLC

RetroFeVLC – The Fast & Powerful Retro Gaming Frontend

RetroFeVLC is a modern, actively developed fork of the popular open-source RetroFE frontend, rebuilt with VLC media library support and optimized for Windows. It delivers a beautiful, highly customizable, and lightning-fast interface for launching and managing your retro game collection across 100+ systems.

Born as an evolution of RetroFE, RetroFeVLC replaces the older GStreamer backend with VLC for superior video, image, and animated media handling. The result is a smoother, more stable, and visually stunning experience — perfect for arcade cabinets, HTPCs, or dedicated retro gaming PCs.

Why RetroFeVLC Stands Out

It combines the deep customization power of classic frontends with modern conveniences and performance improvements that make setup and daily use a joy:

Built-in GUI for easy configuration — No more editing text files for basic settings. Access Menus, Controls, Layouts, and more directly from the interface.

Blazing-fast performance — Menu navigation is quick and responsive, even with large collections.

Stunning 4K visuals — Full 4K support with customizable scale quality options for crystal-clear artwork and videos.

Rich media & animations — Native support for animated GIFs and WebP images, plus major memory leak fixes for rock-solid stability.

Advanced layout tools — Include files, variables, per-game personalized views, GameInfo stats (times played, most played games, etc.), and a clean single-game focus page.

Dynamic playlists — Real-time “All Games”, “All Favorites”, and “All Last Played” playlists that update automatically (via external add-on).

Powerful external add-ons — Including start.exe launcher with playlist management, settings, controls, and quick access to everything. Additional tools like marquee animation control, screen mirroring to marquee, layout creator/editor, menu hierarchy manager, and kiosk mode (launch directly from shell without Windows desktop).

Navigation feels intuitive, layouts are flexible and easy to manage, and adding new systems or screens happens without crashes or headaches.

Quick Start Guide

  1. Download the latest Full All Addons Include version
  2. Extract the archive to any folder (e.g. E:\RetroFeVLC)
  3. Run Start.exe
  4. Right-click the Start icon in the System Tray → RetroFeVLC Options → Settings Editor
  5. Right-click again → Controls Editor
  6. Start adding your own game collections
