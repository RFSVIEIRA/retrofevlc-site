Welcome to RetroFeVLC

RetroFeVLC is a powerful fork of RetroFE that replaces GStreamer with VLC, bringing dozens of new features, a smart addon system, and a much more user-friendly experience.

Key Features

🧠 Start.exe – The Brain

Intelligent launcher with System Tray, automatic sync of Favorites & Last Played, JoyToKey, SuperF4 and more.

🔍 Integrated Search

Lightning-fast game search (F11 by default) with full gamepad support.

🎨 Advanced VLC Layout Engine

Grid menus, per-game layouts, random slideshows, 3D effects, and built-in Options Menu.

📊 Powerful Metadata System

HyperSpin, MAME, truRIP + AI Genre Fixer and Metadata Regenerator.

🖥️ Multi-Monitor & Marquee

Advanced Marquee Monitor Duplicator.

🔧 Visual Editors

MenuManager, Settings, Controls & Layout Editor.
