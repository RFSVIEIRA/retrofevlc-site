Welcome to RetroFeVLC

RetroFeVLC is a powerful fork of RetroFE that replaces GStreamer with VLC, adds dozens of new features, a smart addon system, and a much more user-friendly interface.

Key Features

🧠 Start.exe – The Brain

Intelligent launcher with System Tray, automatic Favorites/Last Played sync, JoyToKey, SuperF4 and more.

🔍 Integrated Search

Fast game search (default F11) with gamepad support – global or per-system.

🎨 Advanced Layout Engine (VLC)

Grid menus, per-game layouts, random slideshows, 3D effects, modular XML and in-frontend Options Menu.

📊 Metadata System

Full support for HyperSpin, MAME, truRIP + AI Genre Fixer and Metadata Regenerator.

🖥️ Multi-Monitor & Marquee

Advanced marquee duplicator and per-monitor settings.

Quick Start Guide

  1. Download the Full All Addons Include package
  2. Extract to any folder (e.g. E:\RetroFeVLC)
  3. Run Start.exe
  4. Right-click tray icon → RetroFeVLC Options → Settings Editor & Controls Editor
  5. Add your game collections

Addons

Start.exe • Search • MenuManager • Metadata Regenerator • GenreFixer (AI) • Story Translator (AI) • Layout Editor • KioskMode • RetroRa Sync • Marquee Duplicator

Layouts & Customization

Extremely powerful layout system: grids, per-game XML, variables, includes, 3D effects, random media, and built-in Options Menu to change themes without editing files.

About the Project

RetroFeVLC is a community-driven fork developed by an enthusiast. Goal: make RetroFE even better while keeping it open-source (GPL3).
